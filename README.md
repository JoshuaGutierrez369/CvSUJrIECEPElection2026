# CvSU Jr. IECEP Elections 2026

Official candidate showcase website for the CvSU Jr. IECEP Elections A.Y. 2026-2027.

Live site: https://joshuagutierrez369.github.io/CvSUJrIECEPElection2026/

## Features

- Official candidate profiles with posters, mottos, and platforms
- Category filters and search
- Mobile-friendly retro election theme inspired by the filing of candidacy materials

## Local preview

Open `index.html` in a browser, or serve the folder with any static file server.
